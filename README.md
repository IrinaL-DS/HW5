**Регрессионный анализ опросника «Короткая темная триада» (SD3)**

Этот проект посвящен исследованию взаимосвязей между тремя чертами личности: нарциссизмом, макиавеллизмом и психопатией. Цель работы — сравнить различные алгоритмы регрессии и выбрать лучший для предсказания психологического профиля.

**Датасет**

Данные предоставлены проектом **OpenPsychometrics.org.** Набор данных получен из открытого архива интерактивных психологических тестов и содержит ответы респондентов на опросник Short Dark Triad (SD3).

* **Источник:** \[Short Dark Triad Responses\] ([https://www.kaggle.com/datasets/lucasgreenwell/short-dark-triad-responses](https://www.kaggle.com/datasets/lucasgreenwell/short-dark-triad-responses)) с платформы Kaggle

* **Файл:** data.csv (разделитель \\t)

* **Примечание:** Файл необходимо скачать с Kaggle и загрузить в среду выполнения (Colab) перед запуском кода.

**Ход исследования**

**1\. Подготовка данных и первая попытка**

На первом этапе было реализовано предсказание шкалы **Макиавеллизма**. Однако результат **R2 ![][image1]\= 1.0** показал «утечку данных» (модель просто складывала вопросы, входящие в саму шкалу). Это было не очень информативно с точки зрения машинного обучения.

**2\. Психопатия с инверсией и честное обучение**

Для получения реальных результатов был изменен подход:

* Выбрана целевой переменной **Психопатия**.

* **Учтена инверсия:** вопросы **P2** и **P7** были пересчитаны по формуле 6 \- x согласно ключу теста.

* **Исключена утечка:** из признаков были полностью удалены вопросы, относящиеся к психопатии. Модель обучалась предсказывать психопатию **только** на основе ответов про нарциссизм и макиавеллизм.

**3\. Сравнение моделей**

Было реализовано и протестировано 7 регрессоров:

1. **LGBM Regressor** (Градиентный бустинг)

2. **Linear Regression** (Линейная регрессия)

3. **Elastic Net** (Линейная с комбинированной регуляризацией)

4. **Lasso** (Линейная с L1-регуляризацией)

5. **Random Forest** (Случайный лес)

6. **AdaBoost** (Адаптивный бустинг)

7. **Dummy Regressor** (Базовая модель для сравнения)

**4\. Личное предсказание**

В проект встроен блок для индивидуального тестирования. На основе введенных ответов по двум шкалам модель предсказывает вероятный балл психопатии пользователя.

**Результаты и выводы**

| Модель | R2 Score | MAE (Ошибка) |
| :---- | :---- | :---- |
| **LGBM** | **\~0.59** | **3.64** |
| Linear Regression | \~0.58 | 3.68 |
| Dummy (Baseline) | \~0.00 | 5.92 |

В таблицу результатов включена модель **Dummy Regressor**, которая всегда предсказывает среднее значение целевой переменной. Это необходимо по трем причинам:

* **Точка отсчета (Baseline):** Он служит эталоном «бесполезной» модели. Если сложный алгоритм (например, Random Forest) показывает результат близкий к результату Dummy, значит, обучение не удалось.

* **Доказательство обучения:** Тот факт, что **LGBM** показал **R2 ![][image1]≈ 0.59** ![][image1], а Dummy \= **0\.0**![][image1], доказывает, что модели действительно выявили скрытые психологические закономерности в данных.

* **Интерпретация R²:** Метрика **R2** по определению равна **0** для модели, которая просто угадывает среднее. Это подтверждает корректность расчетов.

Победителем стал регрессор **LGBM (Light Gradient Boosting Machine)**.  
**Почему он лучший:**

1. **Точность:** Точность модели (**R2 ![][image1]≈ 0.6)** является очень высокой для психологических исследований.

2. **Гибкость:** В отличие от линейных моделей, LGBM способен находить нелинейные зависимости между чертами личности (например, когда один признак усиливает влияние другого только при определенных условиях). Высокие показатели линейных моделей подтверждают гипотезу о том, что черты «Темной триады» тесно связаны между собой прямой корреляцией.

3. **Скорость:** Алгоритм работает значительно быстрее классических деревьев решений на больших объемах данных.

Для проверки качества обучения **LGBM** был построен график сравнения предсказанных значений с реальными результатами теста.

**Интерпретация:**

* Диагональная линия: Идеал, к которому стремится модель.  
* Облако точек: Демонстрирует реальную точность. Тот факт, что облако имеет вытянутую форму вдоль линии, подтверждает высокую корреляцию между предсказаниями и реальностью.  
* Вывод: Модель **LightGBM** успешно справляется с оценкой психологического профиля, допуская минимальные отклонения в центральной части распределения.
